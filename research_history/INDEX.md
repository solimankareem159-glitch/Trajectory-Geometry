## Folder Structure

Each experiment follows a standardized structure:

```
EXP-<NN>_<ShortTitle>_<YYYY-MM-DD>/
├── data/              # Raw and processed outputs
├── analysis/          # Scripts, notebooks, summaries
├── summary.md         # Concise experiment summary
└── metadata.md        # Hypothesis, model, params, date, notes
```
